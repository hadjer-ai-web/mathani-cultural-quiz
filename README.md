# 🕌 مسابقة Mathani الثقافية التفاعلية

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## 📖 عن المشروع

هذا تطبيق ويب تفاعلي يقدم مسابقة ثقافية إسلامية ممتعة من إعداد قناة **Mathani** لحفظ القرآن وتدبره. يتضمن الكويز **15 سؤالاً** متنوعاً في:
- قصص الأنبياء والسيرة النبوية.
- مفاهيم دينية مبسطة.
- أخلاقيات وقيم المسلم اليومية.

تم تصميم الواجهة لتكون سهلة الاستخدام، مع عرض النتائج التفصيلية ومراجعة الإجابات، وإمكانية مشاركة النتيجة على واتساب أو نسخها لتعليقات يوتيوب.

---

## 🚀 الميزات

- ✅ 15 سؤالاً تفاعلياً مع تلميحات مساعدة.
- ✅ عرض النتيجة النهائية ونسبة الإجابة الصحيحة.
- ✅ مراجعة كاملة للأسئلة مع الشرح لكل خيار.
- ✅ مشاركة النتيجة عبر واتساب أو نسخ النص.
- ✅ تصميم متجاوب (جوال، لوحي، حاسوب).
- ✅ شريط تقدم متحرك وإشارات بصرية فورية.
